# React for govuk
Rapid prototyping of govuk sites - ith React

  * React
  * Redux
  * React Router
  * React-swipe
  * ES6
  * Webpack


##Quick Start

```
    npm install
    npm start
```

Goto [http://localhost:3000/](http://localhost:3000/)

Use left and right arrows to navigate the slideshow.
